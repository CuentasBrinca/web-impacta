One-sentence: Labelled dark text input with a magenta focus ring, as used in the pre-registration form.

```jsx
<Input label="Email corporativo" type="email" required placeholder="tu@empresa.cl" />
<Input label="Organización" required />
```

`label` + optional `required` asterisk (magenta), optional `hint` below. Field is a subtle white-alpha fill with hairline border; focus turns the border magenta and adds a soft glow. Forwards all native input props.
