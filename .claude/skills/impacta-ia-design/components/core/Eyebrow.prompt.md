One-sentence: The uppercase magenta eyebrow that labels every section ("El contexto", "Para quién").

```jsx
<Eyebrow>El contexto</Eyebrow>
<Eyebrow accent={false}>Datos preliminares</Eyebrow>
```

Uses FH Lecturis Rounded Bold, wide tracking, uppercase. `accent` (default) shows a magenta short rule + magenta text; set false for a muted neutral variant.
